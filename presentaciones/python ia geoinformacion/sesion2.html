Condicionales: if / elif / else

Python toma decisiones. Evalúa una condición → ejecuta el bloque correspondiente.

condicional.py
# Clasificar un municipio por población poblacion = 531275 if poblacion > 500000: categoria = "Ciudad grande" elif poblacion > 100000: categoria = "Ciudad intermedia" elif poblacion > 20000: categoria = "Municipio mediano" else: categoria = "Municipio pequeño" print(f"{categoria}")
Indentación: Los 4 espacios antes de cada línea dentro del if son obligatorios. Python usa la indentación para saber qué pertenece a cada bloque.

Bucle while: repetir hasta cumplir

A diferencia de for, el while repite mientras una condición sea True:

monitoreo.py
# Monitoreo de temperatura del río temp = 32.0 # grados Celsius umbral = 25.0 hora = 0 while temp > umbral: print(f"⏰ Hora {hora}: {temp:.1f}°C 🔴") temp -= 1.5 # baja 1.5° por hora hora += 1 print(f"✅ Hora {hora}: {temp:.1f}°C — Bajo umbral")
Salida
Monitor de temperatura
Umbral:
25.0°C
for vs while: Usa for cuando sabes cuántas veces. Usa while cuando depende de una condición.

Función Haversine: distancia real

La distancia "en línea recta" sobre la superficie terrestre. Selecciona dos ciudades:

Calculadora Haversine
Distancia Haversine
78.2 km
haversine.py
import math def haversine(lat1, lon1, lat2, lon2): """Distancia en km entre dos puntos""" R = 6371 # Radio de la Tierra dlat = math.radians(lat2 - lat1) dlon = math.radians(lon2 - lon1) a = (math.sin(dlat/2)**2 + math.cos(math.radians(lat1)) * math.cos(math.radians(lat2)) * math.sin(dlon/2)**2) return R * 2 * math.asin(math.sqrt(a)) # Villavicencio → Bogotá d = haversine(4.142, -73.626, 4.711, -74.072) print(f"📏 {d:.1f} km")
¿Por qué Haversine? La Tierra es esférica. La distancia euclidiana (√) que vimos en la Clase 1 es una aproximación. Haversine calcula la distancia sobre la curva terrestre.
